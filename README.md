# Steps to Activate Manually

## Clone the repo

```bash
git clone https://github.com/manishprivet/darkwebwhatsapp.git
```

## Go to Chrome > Settings > Extensions or Type

```bash
chrome://extensions/
```

in the address bar

## Enable Developer Mode

## Click on Load Unpacked

## Select the repo folder

## You're done.
